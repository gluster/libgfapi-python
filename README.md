# Overview

Python bindings for the [GlusterFS](http://www.gluster.org) libgfapi interface

# Installation

1) Clone the git repo

```
$ git clone https://github.com/gluster/libgfapi-python.git
$ cd libgfapi-python
```

2) Run the setup script

```
$ sudo python setup.py install
```
